--- ENG ---

Scene Test in /Scenes serves to test how differect values of density function influence terrain.
In order to work AutoUpdateInPlay must be checked.
Parameters are in the inspector on the right side of the window if default settings are left. First MapGenerator
must be chosen on the left side (Hierarchy window). To see the scene Scene window must be chosen, it is over Unity
window. Test scene's camera is turned off, so nothing is seen in Game window.


To start a game in editor Persistent Scene should be started first - it is a basis for the game.
